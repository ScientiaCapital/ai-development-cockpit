## The Real Reporting Requirements for Large Solar Projects

Based on actual C&I solar EPC best practices:

### Pre-Construction Phase Reports

1. **Site Assessment Report** (Coperniq Form: Solar Site Assessment)
   - Roof/ground condition
   - Electrical infrastructure
   - Utility rate analysis
   - Shade analysis
   - Photo documentation

2. **Design Package** (Coperniq: Attachments + System record)
   - Panel layout
   - Electrical one-line
   - Structural calculations
   - String sizing
   - Production forecast

3. **Permit Status Report** (Coperniq: Project phase tracking)
   - Building permit: Application date, status, approval date
   - Electrical permit: Same tracking
   - Utility interconnection: Application, study status, approval

### Construction Phase Reports

4. **Daily Field Report** (Coperniq Form: Daily Log)
   - Weather conditions
   - Workers on-site (by company)
   - Work completed
   - Issues/delays
   - Safety observations
   - Photos (minimum 10/day for large projects)

5. **Weekly Progress Report** (Coperniq: Auto-generated)
   - % complete by phase
   - Schedule variance
   - Budget variance
   - Open issues
   - Next week's plan
   - Photo summary

6. **Quality Control Report** (Coperniq Forms)
   - Torque verification (module mounting)
   - Wire management inspection
   - Grounding continuity
   - String voltage verification

7. **Safety Report** (Coperniq Forms)
   - Daily toolbox talk log
   - Incident reports
   - Near-miss documentation
   - PPE compliance checks

### Commissioning Phase Reports

8. **System Commissioning Report** (Coperniq Form: Commissioning Checklist)
   - Inverter startup sequence
   - String voltage verification
   - IV curve testing (if required)
   - Monitoring setup
   - Anti-islanding test

9. **Inspection Documentation** (Coperniq: Attachments)
   - Building inspection certificate
   - Electrical inspection sign-off
   - Utility witness test results
   - Fire marshal approval (if required)

10. **As-Built Documentation** (Coperniq: Project attachments)
    - Final panel layout
    - Updated one-line
    - Equipment serial numbers (Coperniq Assets)
    - Warranty documents

### Post-Commissioning (O&M) Reports

11. **Production Report** (Coperniq Systems: Monthly/Quarterly)
    - Expected vs. actual kWh
    - Performance ratio
    - Degradation analysis
    - Production guarantee verification

12. **O&M Visit Report** (Coperniq Forms)
    - Visual inspection results
    - Electrical testing
    - Cleaning performed
    - Issues found
    - Recommendations

13. **Asset Status Report** (Coperniq Assets)
    - Equipment health
    - Warranty status
    - Remaining lifetime
    - Replacement recommendations
