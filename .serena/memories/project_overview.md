# AI Development Cockpit - Project Overview

## Project Purpose
The AI Development Cockpit is a sophisticated **dual-domain LLM platform** that democratizes access to 500,000+ AI models through 97% cost savings compared to traditional APIs. It serves as both:

1. **SwaggyStacks.com** - Developer-focused terminal theme platform
2. **ScientiaCapital.com** - Enterprise-focused corporate dashboard platform

## Tech Stack
- **Frontend**: Next.js 14, TypeScript, React 18
- **Styling**: Tailwind CSS, CSS Modules
- **PWA**: next-pwa, Service Workers
- **Animation**: Framer Motion
- **Authentication**: Supabase with multi-organization support
- **Testing**: Jest (unit), Playwright (E2E), comprehensive chaos engineering
- **Deployment**: RunPod integration, Docker containers
- **Monitoring**: Prometheus, OpenTelemetry, real-time metrics
- **AI Integration**: HuggingFace, vLLM, multiple LLM providers
- **MCP Servers**: Task Master AI, Serena, Shrimp Task Manager

## Architecture
- **Multi-domain routing** with intelligent domain detection
- **Mobile-first PWA** with offline capabilities
- **Microservices architecture** with service isolation
- **Real-time monitoring** and cost optimization
- **Enterprise-grade testing** with chaos engineering
- **CI/CD pipeline** with automated quality gates

## Key Features
- Model marketplace with 500k+ AI models
- Real-time cost estimation and optimization
- Dual-domain A/B testing strategy
- MCP server integration for development workflow
- Comprehensive authentication and RBAC
- PWA capabilities with offline support
- Enterprise monitoring and alerting
